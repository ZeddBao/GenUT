# 自动单元测试生成器应当支持的场景

本文档列出了自动单元测试生成器应当支持的各种测试场景，基于通用单元测试需求、C语言特性以及常见的测试挑战。

## 1. 基础函数测试

### 1.1 简单函数测试
- **纯函数**：无副作用，仅基于输入返回输出
- **无参数函数**：测试全局状态变化或常量返回值
- **void函数**：测试副作用和状态改变

### 1.2 参数类型测试
- **基本类型**：int, float, double, char, bool等
- **指针类型**：参数指针、返回指针
- **数组参数**：定长数组、指针+长度参数
- **结构体参数**：值传递、指针传递
- **枚举类型**：所有枚举值覆盖
- **联合体参数**：不同成员访问路径
- **位字段参数**：位操作测试

### 1.3. 参数组合测试
- **边界值分析**：最小值、最大值、刚好超出
- **等价类划分**：同一等价类代表性值
- **组合测试**：多参数组合覆盖
- **正交数组**：减少测试用例数

### 1.4 返回值测试
- **正常返回值**：各种边界值和典型值
- **错误码返回**：负数、特定错误码
- **NULL指针返回**：分配失败场景
- **布尔返回值**：true/false分支

## 2. 控制流测试

### 2.1 条件分支覆盖
- **if/else语句**：所有条件分支
- **嵌套if语句**：多层条件组合
- **switch语句**：所有case分支，包括default
- **三元运算符**：? : 表达式

### 2.2 循环覆盖
- **for循环**：零次、一次、多次迭代
- **while循环**：条件边界测试
- **do-while循环**：至少执行一次的场景
- **嵌套循环**：内层外层组合
- **break/continue**：循环控制语句

### 2.3 复杂逻辑表达式
- **逻辑运算符**：&&, ||, ! 组合
- **比较运算符**：==, !=, <, >, <=, >=
- **算术运算**：+, -, *, /, % 在条件中
- **位运算**：&, |, ^, ~, <<, >> 在条件中

## 3. 数据结构测试

### 3.1 结构体测试
- **嵌套结构体**：多层字段访问
- **结构体指针**：-> 操作符访问
- **结构体数组**：数组元素访问
- **匿名结构体**：内嵌结构体访问
- **位字段结构体**：位操作测试

### 3.2 数组测试
- **一维数组**：索引边界测试
- **多维数组**：行列访问
- **动态数组**：指针+长度参数
- **字符数组**：字符串操作

### 3.3 指针测试
- **单级指针**：NULL和非NULL场景
- **多级指针**：**pp 双重间接
- **函数指针**：回调函数测试
- **void指针**：泛型数据处理
